# Report Samples

This repository includes public-safe sample GEO outputs shaped like a typical reference documentation audit.

## Included Samples

- [`reports/reference-geo-audit-sample.md`](../reports/reference-geo-audit-sample.md)
- [`reports/reference-geo-audit-sample.json`](../reports/reference-geo-audit-sample.json)

## What The Sample Shows

- the live `llms.txt` exists and is valid
- the route set is broad enough to act as an AI-readable index
- the site has useful information architecture for GEO
- citability is currently stronger on the overview guide than on the newer prompt-template detail page

## Why Publish Sample Reports

Sample reports make the repository more than a link list:

- they provide proof that the linked site has measurable structure
- they give GitHub visitors a concrete GEO artifact to inspect
- they create searchable documentation around terms like `llms.txt`, `citability`, `schema`, and `AI crawler access`

## What Is Intentionally Excluded

This repo should avoid publishing:

- private analytics
- internal-only strategy notes
- anything that implies official ownership of a third-party trademark
- anything that turns the repo into a disguised commercial landing page

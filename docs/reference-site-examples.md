# Reference Site Examples

This document lists **illustrative** routes when applying this repository’s patterns. URLs use the IETF-reserved domain `example.org`; **replace the host and paths** with your real public documentation site.

Links below are **canonical** (no referral or campaign parameters). For GitHub or analytics tagging, define that separately—for example in [`docs/measurement-plan.md`](measurement-plan.md)—and do not treat those tags as part of the route map in [`examples/reference-route-map.md`](../examples/reference-route-map.md).

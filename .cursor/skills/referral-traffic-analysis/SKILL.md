# Referral Traffic Analysis

This skill covers the import-to-report workflow for referral traffic learning.

## Use When

- You have GA4 or CSV referral exports to analyze.
- You want to learn which external sites send useful traffic.
- You need reusable patterns instead of one-off traffic observations.

## Core Workflow

1. Validate the input format and schema.
2. Import and normalize source and landing-page data.
3. Analyze traffic patterns and segment notable sources.
4. Extract reusable insights and write a learning report.

## Typical Outputs

- Normalized referral datasets
- Pattern analysis
- Reusable traffic insights
- Referral learning reports

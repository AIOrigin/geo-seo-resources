# Site Structure SEO Analyzer

This skill focuses on turning a site review into actionable structure and SEO findings.

## Primary Modes

- `audit`: inspect headings, metadata, schema, linking, and obvious SEO gaps
- `media`: inventory reusable images, videos, and assets
- `template-mapping`: map a site into a repeatable landing-page structure
- `landing-plan`: turn the audit into an implementation or expansion brief

## Use When

- You need a structured review of a target page or competitor page.
- You want to compare route structure, content depth, and reusable media patterns.
- You want to translate a reference site into a landing-page execution brief.

## Deliverables

- SEO audit findings
- Content and media inventory
- Template or page-structure mapping
- Landing-page implementation notes

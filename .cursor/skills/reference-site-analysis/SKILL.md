# Reference Site Analysis

This skill covers structured reference-site learning and comparison.

## Use When

- You want to study competitor or reference sites.
- You need screenshots, structure notes, and comparison findings.
- You want to extract reusable landing and SEO patterns from other sites.

## Core Workflow

1. Visit and capture each reference site.
2. Save notes, screenshots, and raw structure when needed.
3. Compare the references with the target landing.
4. Turn findings into clone or improvement guidance.

## Typical Outputs

- Per-site structure notes
- Comparison documents
- Screenshots and saved references
- Clone or improvement recommendations

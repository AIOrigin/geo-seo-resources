# SEO Tracking System

This skill covers the shared tracking and dashboard layer for SEO workflows.

## Use When

- You need to record agent runs and workflow steps.
- You want a dashboard for trends, screenshots, or performance metrics.
- You need reporting across multiple SEO workflows instead of isolated logs.

## Core Functions

1. Track workflow runs and detailed actions.
2. Record trend lifecycle and content status.
3. Store screenshot and asset metadata.
4. Expose dashboard and reporting views for SEO work.

## Typical Outputs

- Tracking database records
- Dashboard summaries
- Run history and action logs
- Trend and performance reports

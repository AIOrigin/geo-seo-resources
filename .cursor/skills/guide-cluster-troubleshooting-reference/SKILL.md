# Guide Cluster Troubleshooting Reference

This skill covers the support and reference layer of a guide hub.

## Use When

- You need support-style content that answers recurring questions.
- You want stronger reference coverage for AI-readable discovery.
- You want a stable cluster for long-tail search intent and internal support links.

## Working Rules

1. Keep definitions and fixes explicit and scannable.
2. Prefer issue-resolution structure over generic overview prose.
3. Use this cluster to support both human readers and machine extraction.

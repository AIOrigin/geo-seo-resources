# Guide Cluster Getting Started

This skill covers the introductory guide cluster at the top of a guide hub.

## Use When

- You need start-here pages for new visitors.
- You want to check if the introductory guide set is complete and internally linked.
- You want to map awareness-stage search intent to the right guide pages.

## Working Rules

1. Keep these pages introductory, high-clarity, and easy to cite.
2. Treat them as the top of the guide funnel.
3. Use them as anchor destinations from your guide hub and related internal links.

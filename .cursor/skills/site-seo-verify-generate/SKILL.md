# Site SEO Verify Generate

This skill is a reusable end-to-end SEO workflow, not a project-specific name.

## Workflow Stages

1. `verify`: check current route health, metadata, crawlability, and content coverage
2. `analyze`: identify route gaps, keyword opportunities, and priority targets
3. `generate`: prepare new or improved capability, guide, or prompt-template content
4. `deploy`: validate readiness and hand off for publishing

## Use When

- You want an end-to-end SEO workflow instead of a single page edit.
- You need to turn audit findings into prioritized new content.
- You want a repeatable handoff from SEO review to route expansion.

## Outputs

- Audit notes
- Gap analysis
- Content priority queue
- Route-specific generation briefs

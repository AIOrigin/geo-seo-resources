# Guide Cluster Comparisons Decision

This skill covers decision-stage guides such as comparisons and use-case pages.

## Use When

- You need decision-stage pages for people comparing tools or use cases.
- You want to audit neutral comparison coverage and route intent.
- You need a cluster for bottom-funnel informational queries.

## Working Rules

1. Keep comparisons neutral, route-aware, and evidence-driven.
2. Separate use-case explanation from direct product claims.
3. Maintain clean links back to introductory and workflow pages.

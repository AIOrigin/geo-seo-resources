# Capability Route Cluster

This skill represents a capability-style route family as a reusable cluster.

## Use When

- You need the canonical list of capability routes.
- You want to prioritize or expand the capability cluster as a group.
- You need a reusable brief for capability-page SEO, GEO, or internal linking.

## Working Rules

1. Treat the capability family as a topic cluster, not as isolated pages.
2. Keep per-page metadata and multilingual coverage aligned for the highest-priority items.
3. Prefer implemented route taxonomies over older planning docs.

# SEO Growth Workflow

This skill covers the growth-operations layer that turns discovery into queued execution.

## Use When

- You need to turn discovery findings into actionable jobs.
- You want page briefs, link suggestions, or payload packs.
- You need to inspect blockers across the SEO growth pipeline.

## Core Workflow

1. Choose the correct job family.
2. Check dependency gates before promising end-to-end execution.
3. Convert discoveries into briefs, payloads, or queue items.
4. Keep date-based outputs aligned across the pipeline.

## Typical Outputs

- Opportunity-board jobs
- Page briefs
- Internal-link suggestions
- Channel or payload packs

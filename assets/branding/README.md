# Branding Assets

This folder stores repository-level branding assets for `geo-seo-resources`.

## Files

- `icon.svg`: canonical repository icon (editable vector source)
- `icon-generated-1024.png`: raster icon (1024×1024) produced with image generation from `docs/branding-icon-spec.md`; use for GitHub org/repo avatar or as reference when redrawing SVG
- `social-preview.svg`: repository social preview base asset (1280×640, 2:1). Embedded at the top of each `README*.md` on GitHub; includes `<title>` / `<desc>` for accessibility when the SVG is opened or inlined. The README banner links here for asset notes.

## Rules

- Keep branding generic to SEO and GEO workflows
- Do not make the icon resemble an official third-party product logo you do not own
- Prefer SVG as the editable source of truth

See `docs/branding-icon-spec.md` for the design direction, palette, and usage rules.
